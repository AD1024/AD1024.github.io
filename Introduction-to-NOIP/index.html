1 C语言基础知识与易错点整理

基础知识:C语言基础语法-中文版

1.1 数据输入

1.1.1 从屏幕读入

C语言从屏幕读取数据的函数是scanf(); 其中注意：输入除字符串(char* 与 char[])类型时需要在要赋值的变量前加’&’ 即(输入十进制整数为例)：

int x;
scanf("%d",&x);

当输入字符串(char数组)时，不需要加’&’。即:

char str[1001];
scanf("%s",str);

不建议使用gets()函数读入字符串。

1.1.2 从文件读入

很多考试要求使用文件输入、输出。在C语言中，我们可以通过使用freopen()来进行输入/输出重定向。其使用方法是：

freopen(文件名,读入/读出,stdin/stdout);

如:

freopen("test.in","r",stdin);
freopen("test.out","w",stdout);

提示：在输出答案之后，退出程序前建议加上:

fclose(stdin);
fclose(stdout);

1.2 输出数据

输出数据在做题中是至关重要的一步，有时你可能因为输出格式不对但是答案完全正确而导致爆0.为避免这个情况，请仔细阅读题目输出格式要求。

输出数据中的易错点:

  1. 若输出多行，行末是否保留空格；
  2. 输出数据的大小写；
  3. 是否输出到文件；
  4. 输出数据的缩进/空格数量。

1.3 数据范围

读完题之后，不要立刻去做题，一定要看一下数据范围。一是确定使用什么样的算法解决这个问题，由于每种方式时间复杂度和空间复杂度不相同，所以看数据范围是十分重要的；二是确定开多大是数组，或者使用什么样的数据结构存储输入数据。

各个数据范围对应时间复杂度:
小于等于8 N^N
小于等于11 N!
小于等于25 2^N
小于等于300 N^3
小于等于3000 N^2
小于等于100W NlogN
小于等于9000W N
小于等于1E logN
任意大小 1

1.3.1 动态内存分配

C语言中可以使用malloc()动态分配内存(C++是new)。不过不推荐使用该方法分配内存，因为手动分配内存是在堆上开空间，而不是栈上，所以程序在执行过程中不会自动释放内存。因此需要使用free(C++为delete/delete[])手动释放空间。

注意：malloc返回的是void*，因此需要进行强制类型转换。

例：动态分配大小为100的int数组

// C语言
int *a=(int*)malloc(sizeof(int)*100);
// 释放内存
free(a);
//-----
// C++
int *a = new int[100];
delete[] a;

注：malloc与new适用于所有类型的内存分配，包括自定义结构体。

1.3.2 数组初始化

C语言中提供了memset()函数初始化数组值，使用方法为:

memset(数组名,初始化值,sizeof(数组名));

注意：在函数外声明的数组默认值为0，但在函数内声明的数组，其数据是随机的数字因此如果在函数内定义数组并使用，不要忘记在使用前初始化它

if(x&1){
  // 是奇数
}else{
  // 是偶数
}

int GCD(int x,int y){

float rsqrt(float x){ // 原理:牛顿迭代
    float x2=x*0.5F;
    int i=*(int*)&x;
    i=0x5f3759df-(i>>1);
    x=*(float*)&i;
    x=x*(1.5F-(x2*x*x)); // 迭代

    return x;
}

3 字符串

3.1 字符串初步

3.1.1 什么是字符串

字符串就是一串字符组成的值。其中可以包括字母大小写，各国语言，数字等符号。我们常用的就是大小写字母以及数字。C语言中，我们常用char str[N] (char型数组)存储字符串类型(N为定值)。STL中的string类型也可以存储字符串数据。

注意：字符串数组起始下标为0，因此strlen()所返回的值是最大下标+1

例如：若字符串a为”123456”，则a[0]=1 a[5]=6 strlen(a)返回6

因此在编写程序时注意取下标时越界的问题

3.1.2 字符串基本操作

对于字符串变量，我们通常有:1. 获得长度 2. 获得某位上字符的值 3. 修改某位上字符的值。

// 获得一个字符串的长度
int length=strlen(str); // strlen()函数在<string.h>中

// 获得第i位的值
char c=str[i];

// 修改第i位的值
str[i]=c;

3.1.3 字符串的遍历

对于字符串类型，我们可以使用游标(cursor)或者指针(pointer)两种方式进行遍历，多数情况我们使用前者，因为比较好控制程序流程。
游标遍历:

char str[1001]={"123456"};
for(int i=0;i<strlen(str);++i){ // 起始下标 0 到 终止下标 strlen(str)-1
  printf("%c",str[i]);
}

3.1.4 字符串比较

对于字符串比较，C语言提供了strcmp()函数(使用方法在网上自行搜索)。这里我们可以手动实现对字符串的比较(是否相同)。

  1. 比较长度-若长度不相等则字符串一定不相等(无空格)
  2. 遍历字符串进行判断
    bool isSame(char a[],char b[]){
int la=strlen(a);int lb=strlen(b); // 获得长度
if(a!=b) return false; //若不相同直接返回
else{
 for(int i=0;i<la;++i){ // 最大下标为la 或 lb (因为二者相等)
   if(a[i]!=b[i]) return false; // 如果同一位置不相等，直接返回
 }
 return true;
}
}

3.2 字符串相关技巧与算法

3.2.1 字符转数字

事实上，字符类型是特殊的int类型，因此他们之间可以进行转化(详见ASCII表)。对于字符类型的x(0-9)，我们可以通过x-‘0’将x所对应的int计算出来。这时因为在ASCII表中，0-9是连续的，因此可以通过”-‘0’”把对应值算出来。

3.2.2 字符串转整数

有时我们会遇到字符串转换整数的问题，这时我们需要打出10^N的表，然后进行逆向遍历(从个位开始)计算出结果。

const int Pow[9]={1,10,100,1000,10000,100000,1000000,10000000,100000000,1000000000};
int str2int(char str[]){
  int cur=0,ret=0;
  for(int i=strlen(str);i>=0;--i){ // 逆向遍历
    ret+=(str[i]-'0')*Pow[cur++]; // 每次乘完cur自动+1
  }
}

3.2.3 字符串倒转

此方法同样适用于判断回文。使用对折反转法，复杂度O(N/2)。

int N = strlen(str);
for(int i=0;i<(N>>1);++i){ // N>>1 = N/2 参考上文
  swap(str[i],str[N-i-1]); // swap即可使用上文中的交换实现
}

3.2.4 整数转字符串

相对3.2.2的反操作，但是比3.2.2容易，我们只需要不断对10取模即可，最后我们需要反转数组(因为最高位从个位开始)。

void int2str(int x){
  char str[1001];
  int cur=0;
  while(x!=0){
    int i=x%10; // 获得最低位
    str[cur++]=i+'0'; // 加入字符串(由于int需要转换到ASCII码格式，因此需要+'0');
    x/=10; //移除最高位
  }
  reverse(str); // reverse方法可参考3.2.3
}

4 模拟算法

事实上，模拟不能被称作算法，因为模拟仅仅是使用代码模拟一些事情的发生过程。但是好的模拟能够直接体现出一个人的语言功底，因此不可轻易忽视。

4.1 模拟算法的技巧

模拟也有很多的技巧，如使用HASH去重(目录)，记录访问历史等。这里会介绍HASH的使用方法。

4.1.1 HASH去重

看着名字很高端，其实很lowB。Hash去重就是开一个数组记录某个点/元素/位置/blablabla是否使用过，或使用值是多少。这里拿一个题举例子:
数字统计
这个题就可以使用该方法做。具体处理方法:开一个hash数组记录各个数字使用次数。

// 核心代码
int ans[100001];
memset(ans,0,sizeof(ans)); // 全部初始化为0
for(int i=0;i<strlen(str);++i){
  ans[str[i]-'0']++; // str[i]-'0'所对应位置的数+1
}

4.2.2 二维数组记录位置信息

在某些模拟中，我们需要记录对于一些坐标是否访问过的记录。此时我们可以使用布尔型二维数组记录某个位置是否被访问过。

bool vis[1001][1001];
memset(vis,false,sizeof(vis)); //初始化访问记录数组
vis[x][y]=true; // 坐标或状态(x,y)被访问过

当然，我们可以在数据允许的范围内开更多维的数组记录更多数据，如：

bool vis[100][100][100];
memset(vis,false,sizeof(vis)); // 初始化访问数组
vis[x][y][value]=true; // 在状态或坐标(x,y)上，值为value的情况已经被访问过

5 递归与搜索算法

5.1 深度优先搜索

5.1.1 递归枚举

递归枚举，属于搜索算法的基础(属于深度优先搜索的一种)，其实现方法为C语言中的递归。递归枚举可以称为“一路走到黑”，因为它会在一次搜索中枚举所有状态直到搜到符合题目条件的状态。在编写深度优先搜索时首先确定搜索的边界条件，然后确定需要枚举哪些参数。
一个标准深度优先搜索的框架:

void f(搜索所需参数){
  if(达到边界){
    // 处理结果
  }else{
    for(枚举状态){
      f(下一个状态的参数)
    }
  }
}

例题:给定数字0-9，每个数字可以通过一个路径到达比它大1或者大2的数。如:在1的位置可以到达2或3；在2的位置可以到达3或4….给定整数N(0<=N<=9)，输出所有从0到N的路径。
边界:位置达到N；
状态:当前位置；
下一个状态:当前位置到比它编号大1或大2的位置。

// 核心代码
void f(int k,int idx){
  if(k==N){
    // 输出当前合法路径
  }else{
    a[idx]=k;
    f(k+1,idx+1);
    f(k+2,idx+1);
  }
}

5.1.2 回溯搜索

回溯搜索建立在递归基础之上，属于深度优先搜索。比暴力枚举高端的地方在：它可以记录重复状态，避免一些重复的状态搜索。
例题:求给定数字1-N,问在不重复使用同一个数的前提下，用K个数字加和为素数的方案数有多少。
