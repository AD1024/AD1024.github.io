1 基础语法

1.1 C语言基本数据类型

C语言基本数据类型有:

类型名称 数据类型 范围
char 字符 -128~127
int 整数 -2^32~2^32-1
float 单精度小数 -2^128~2^127
double 双精度小数 -1.0710^308~1.0710^308
bool 布尔值 true~false(0~1)

1.1 声明与赋值语句

1.1.1 声明

在变量使用之前，我们需要声明变量。也就是我们告诉计算机，让它给你准备空间存放这个变量。如：

/*声明3个变量名称为a b c*/
int a;
int b;
int c;

1.1.2 赋值语句

与数学表达类似，C语言中也使用’=’进行变量赋值。如：

int a; // 声明变量
a=1;   // 赋值变量

通过以上语句，我们就将a赋值为1。当然，我们可以将声明与赋值合并在一起:

int a=1;

这样我们就在声明时直接将a赋值为1。

1.2 运算符

在C语言中，运算符有数学中的加、减、乘、除和取模(mod)。在C语言中对应为:

加: +
减: -
乘: *
除: /
取模: %

注意C语言中没有乘方(小角标)，阶乘(!)，根号运算符

1.2.1 变量之间的运算

int a=1;int b=2;
int c=a+b;

通过以上代码，我们将a+b的结果存储在c中。其他运算以此类推

1.2.2 C语言特有运算符

为了方便程序猿，C语言中有一些独有的，数学中没有的运算符：

a++ a– a+=2 a-=2 a*=2 a/=2
对应的意思是:a自增1 a自减1 a自增2 a自减2 a自乘2 a自除2

当然还有

++a –a

其与a++ a–的区别是:运算符在前的先自增/减1再带入运算；运算符在后的，先带入运算再自增/减1。如:

int a=1;
int b=a++; // 此时b=1
int c=a; // 此时c=2
a=1;
b=++a; // 此时 b=2

1.2.3 比较运算符

比较运算符有:
大于 >
小于 <
大于等于 >=
小于等于 <=
是否相等 ==

1.2.4 逻辑运算

逻辑运算中与数学中的逻辑运算相同，有与(并且)，或(或者)，非(取反)。在C语言中表现为:

&& || !

当遇到’&&’时，只有前后两个条件都是真时，才表现为真；
如:

1>2 && 2<3 1="" 假="">2 && 2>4 // 假
1>0 && 2<10 // 真

当遇到’||’时，只有前后两个条件至少一个是真时，才表现为真；
如：

1>2 || 2<0 1="" 假="">2 || 2>1 // 真
2>1 || 1==1 // 真

当遇到’!’时，该运算符后方的命题真实性翻转
如：

!(1>2) // 真
!(2>1) // 假

1.3 选择结构

1.3.1 if结构

在C语言中，可以对一些情况作出判断。其判断只存在对于错，不存在中间值。我们可以使用if()对括号内的情况作出判断。一个if选择为：

if(){
  // 代码块
}

若括号中的条件为真，则会执行代码块；若为假则不执行。
如：

int a=1;int b=2;
if(a>b){
  a-=b;
}

以上代码，if中的语句不会执行。

1.3.2 if-else结构

当然，当条件不为真时，我们也想搞事情，怎么办呢？那么我们就搞一个if-else结构。该结构基本结构如下:

if(){
  // A
}else{
  // B
}

当括号中为真时，则执行A处代码，否则执行B处代码。
if-else结构是可以嵌套的:

if(){
  if(){

  }
}else{
  if(){

  }else{

  }
}

注意else后方只能再写if或者代码，不能继续else
通过这个方法，我们可以做一些复杂状况的判断。

1.3.3 switch-case结构

考虑如下问题：某个班级有着森严的等级制度，颜值越高的同学等级越低，对应等级如下(递减):
颜值 等级
1 A
2 B
3 C
4 D
现在告诉你全班每个人的颜值，请确定同学们的等级。
如果用if-else结构，程序是这样的:

if(){

}else if(){

}else if(){

}else if(){

}

这看着好难看，在班里等级一定很高。所以我们要使用颜值高的switch-case结构。一个switch-case选择结构是这样的:

switch(x){
  case 值1://A;break;
  case 值2://B;break;
  case 值3://C;break;
  default://C;break;
}

当x==值1时，会执行A部分代码；x==2时执行B部分….
default是指，x和值1值2值2都不相等时，执行C后直接退出(当然A,B,C视情况而定)。
使用以上代码，颜值高但是等级低。这是因为switch-case结构仅能判断是否相等，而不能判断其他关系。

1.4 循环结构

为了实现更为复杂的功能和算法，C语言支持循环结构。在C语言中，循环结构有3种：

  • while循环
  • for循环
  • do…while结构
    循环结构可以嵌套

1.4.1 什么是循环结构?

循环结构及重复做同一件事情。在生活中处处存在着循环结构：你每天都奔走在各个教室之间，这是循环；老师判作业时一张一张卷子翻，这也是循环。同样在C语言中，循环是一个很重要的部分，因为它可以为你节省很多时间：计算1+2+3+…+100，如果不使用求和公式你需要手打100个数字；然而使用循环结构，仅仅几行就能实现。

1.4.2 while循环结构

一个while循环结构的基本框架如下:

while(<condition>){
  <code>
}

也就是说，在condition成立的情况下，程序将一直执行code，其中condition可以是任何逻辑判断。
E.g.计算1+2+…+1000;

int x=1;int sum=0;
while(x<=1000){
  sum+=x;
  x+=1;
}

1.4.3 for循环结构

一个for循环的基本结构如下：

for(<initial-condition>;<terminate-condition>;<operation>){
  <code>
}

initial-condition是循环初始条件/初始化；terminate-condition是循环终止条件；operation是一次循环结束后对初始条件作出的变化。
E.g.计算1+2+…+1000

int sum=0;
for(int i=1;i<=1000;i++){
  sum+=i;
}

1.4.4 do…while循环结构

do-while结构在实际应用中并不常见(因为用while是可以替代的)。一个do-while的基本框架如下:

do{
  <code>
}while(<condition>);

与while循环相似，do-while结构的执行条件依旧是condition，而与while循环不同的是在循环开始前，code中的内容会先被执行一次。

1.4.5 continue语句与break语句

continue语句用于跳过一次循环进入下一次循环；
break语句可用于忽视终止条件退出循环。

注意:在执行continue语句后，初始状态依旧会按照编写的规则变化

1.4.6 循环的嵌套

在本小节开始时曾说过，循环是可以嵌套的。那么循环嵌套是什么呢？其实很简单——循环里跑循环。如：求1+..+10,2+..+10,3+..+10 … 10+10的和

int sum=0;
for(int i=1;i<=10;++i){
  for(int j=i;j<=10;++j){
    sum+=j;
  }
}

使用该方法可以简单实现更加复杂的循环结构。

1.5 数组

1.5.1 什么是数组?

数组就是一组数据，可以集中存储一些互相有关联的，或者是需要集中化存储的数据。相比起单独的一个变量，数组更加具有灵活性。数组不仅在算法竞赛扮演着重要的角色，在软件项目开发中也有着至关重要的作用。

1.5.2 一维数组的声明

数组声明事实上和普通的一个变量声明相似，只不过多一个小尾巴：

int a[1001]; //这就是一个数组

数组后面中括号和数字表示：这个变量是个数组，一共可以存下1000个数字。

1.5.3 一维数组的赋值与取值

对于一个数组，我们可以通过使用中括号+下标的方式取值/赋值:

int a[1001];
a[2]=1;

以上代码，将数组a的第二位赋值为1。需要注意的一点是:
C语言中数组起始下标为0
也就是说第二位事实上在这个数组里是第三位。(0->1->2)

同样地，我们也可以通过使用这个方法取值:

int x;
int a[1001];
a[0]=1;
x=a[0];

通过以上代码，我们将x的数值赋值为数组a第一个元素的值。

1.5.4 多维数组的声明与赋值

与一维数组类似，数组可以是多维的(如：二维就是矩阵)。这取决于数组标识符后有几个中括号。例如一个二维数组声明就是:

int a[1001][1001];

表示这是一个10011001的数组(矩阵)。
当然我们不要把“维”这个东西想的太高端，因为这个“维”仅仅是一个概念，实际应用时不需要管那么多。不过需要注意的是空间限制。大小为1000的一维数组占用空间是1000，而二维的就是10001000。维越高，越占用空间。

1.5.5 数组的初始化

数组初始化需要include一个头文件叫做”string.h”(C++为cstring)。其中有一个函数为”memset”，它的作用就是初始化数组(几维数组都可以)。memset使用方法如下:

int a[1001];
memset(a,0,sizeof(a));

传入a的地址，初始化值以及初始化长短；二维数组与其相同
不过注意：memset()是按字节进行，也就是说以下代码:

int a[1001];
memset(a,1,sizeof(a));

是错误的，因为每次赋值只赋值1B，无法保证所有值都是1。

1.6 指针

指针在C语言中属于较为苦难的一部分，因为各种引用关系比较复杂。因此这里仅讲解算法竞赛中会使用到的指针的知识。

1.6.1 声明指针类型

事实上指针类型基于现有的数据类型，如：我要声明一个int类型的指针，我只需要:

int *a;

即可。也就是在标识符前加“*”。其他类型以此类推。

1.6.2 指针类型的作用

指针类型不真正存储数据，而是指向真正存储数据变量的地址，因此对指针类型的操作不会体现在指针类型上，而是会体现在它所指向的变量上。如:

int a=1;
int *p=&a; //&操作符用于取出变量对应的内存地址
*p=10;
/*此时a的值已经变成了10*/

*1.6.3 引用类型

引用类型为附加内容，可跳过，但建议阅读
在C++中还有一种类型叫做引用类型，与指针类型相似。引用类型方便在:如果在其他函数中对形参的修改想要体现在调用处，我们需要用引用类型。引用类型应用如下:

void SWAP(int &a,int &b){
  a^=b^=a^=b;
}

int main(){
  int a=1,b=2;
  SWAP(a,b);
  return 0;
}

使用这个方法，可以在SWAP函数中交换a，b的数值，但是这个修改也会体现在main函数中。

1.7 结构体

为了增强程序对数据的封装性，C语言支持结构体对数据进行集散化。
结构体支持数组

1.7.1 定义结构体

一个结构体的定义框架如下:

struct <Name>{
  Data a;
  Data b;
  ...
};

其中为结构体名称。
例如我想要存储一个人的基本信息:

struct Person{
  int age;
  char name[30];
};

在使用这个结构体时与使用基本数据类型方法一样:

Person me; //声明一个Person结构体数据类型

1.7.2 结构体中数据的修改

有了结构体，我们还需要往里面存一些数据，如何存呢?我们引进”.”和”->”操作符。对于普通数据对象我们使用”.”操作符取数据，对于指针类型我们使用”->”操作费取数据。如:

Person a;
a.age=1;
//----------
Person *p=&a;
p->age=a.age+10;

通过这个方法我们可以修改/访问结构体中的数据。

1.7.3 *结构体的构造函数

* 这部分内容属于附加内容，可跳过，但是推荐阅读
构造函数就像一个工厂，放入原料，出来产品。struct的构造函数只要传入你想要初始化的参数，它会“造出来”一个结构体给你。如:

struct Person{
  int age;
  string name;
  Person(int Age,string Name){ //构造函数(参数不要和结构体里的重名)
    age=Age;
    name=Name;
  }
};

注意:你会发现如果这样写了构造函数，直接定义会编译报错，这是因为如果不写构造函数，其默认的构造函数是没有任何参数的。因此如果想要保留直接定义的支持性，需要写一个空的构造函数:

struct Person{
  int age;
  string name;
  Person(){} //直接定义的支持
  Person(int Age,string Name){ //构造函数(参数不要和结构体里的重名)
    age=Age;
    name=Name;
  }
};
